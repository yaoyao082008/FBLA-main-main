How to use our Calculator ...

To add a class, click the green "Add" button next to the bottom most class.

To remove a class, click the red "Remove" button next to the class.

    Class Name: To enter the class name, enter the name in the input placed under the tag: "Class Name" (unique)
    Class Type: To enter the class type, choose from the dropdown placed under the tag: "Class Type"
    Class Grade: To enter the class grade, enter the grade in the input placed under the tag: "Grade"
